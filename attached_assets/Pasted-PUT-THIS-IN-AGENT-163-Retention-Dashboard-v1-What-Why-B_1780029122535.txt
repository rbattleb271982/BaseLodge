--------------------------------------------------
SECTION 1 — RETENTION KPIs
--------------------------------------------------

Create KPI cards for:

1. DAU
Formula:
last_active_at >= NOW() - 1 day

2. WAU
Formula:
last_active_at >= NOW() - 7 days

3. MAU
Formula:
last_active_at >= NOW() - 30 days

4. DAU / MAU
Formula:
DAU ÷ MAU × 100

Display as %

5. Returning L30

Users active in last 30 days AND created_at < NOW() - 30 days

Each KPI card should display:

- Current value
- Delta vs prior comparable period
- Small sparkline
- Short explanatory subtext

WAU delta:
Current 7-day window vs previous 7-day window

MAU delta:
Suppress delta if prior window = 0
