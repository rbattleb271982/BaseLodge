PART 3 — Trip KPI restructuring

Rename:
- "Trips This Month"
→ "Trips Planned This Month"

Add:
- Trips Planned This Week
- Total Trips Planned

Definitions:
- Trips Planned This Week:
  trips created in trailing 7 days
- Trips Planned This Month:
  trips created this calendar month
- Total Trips Planned:
  all-time trip count

Important:
These are planning-behavior metrics, not trips occurring/traveling this month.

Dashboard goal:
Show whether users are actively planning ski trips.
