- Softer muted taupe / warm gray
  - use: #8F8477 or similar
- Line height should feel elegant and relaxed
- Keep copy width controlled so it wraps cleanly

- Rounded pills or softly rounded tiles
- Soft neutral fill, slightly darker than page background
  - use something close to: #ECE7DE
- Border should be very subtle
  - use: 1px solid #DED6CA
- Unselected options should feel quiet and low-pressure
- Selected options should have:
  - a softer tinted fill

  - slightly stronger text contrast
- No loud chips, no saturated fills, no playful UI treatment

Progress indicator:
- Minimal and refined
- Inactive steps should be very light / soft
- Active step should be slightly darker and more defined
- Keep it understated, not dashboard-like

Buttons:
- Primary button should be a rich maroon tone
  - use: #7A1E1E
- White text
- Pill-shaped
- Slightly refined height, not chunky
- Medium font weight, not too bold
- Full-width
- No heavy shadows
- If any shadow exists, keep it extremely subtle

Icons:
- Remove ALL emojis
- If icons appear anywhere, they must be thin outline icons only
- Match the style of the app’s bottom navigation icons
- Subtle, quiet, editorial
- Never decorative or childish

Spacing:
- Increase breathing room between title and subtext
- Increase breathing room between subtext and choices
- Keep tighter spacing between related controls
- Avoid evenly distributed spacing everywhere
- The screen should feel composed, not mechanical

Tone:
- Premium
- Slightly social
- Not startup-y
- Not gamified
- Not “fun app”
- Not corporate
