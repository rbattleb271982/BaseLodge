3️⃣ EQUIPMENT — CANONICAL SYSTEM (CRITICAL)
Source of truth
Use the existing EquipmentSetup model.
Do NOT add equipment fields to User.
All gear entered via pop-ups must create or update the user’s active EquipmentSetup record.
Required / Optional Fields (on EquipmentSetup)
Add nullable columns if missing:
equipment_status     # own / rent / both
equipment_brand      # skis / board
equipment_model
boot_brand
boot_model
Equipment Step Completion (DERIVED — DO NOT STORE)
The Equipment step is complete if:
An active EquipmentSetup exists for the user AND
equipment_status is NOT null
Brand/model fields are optional and never gate completion.
Example SQLAlchemy — Create / Update EquipmentSetup
equipment = EquipmentSetup.query.filter_by(
    user_id=current_user.id,
    is_active=True
).first()

if not equipment:
    equipment = EquipmentSetup(
        user_id=current_user.id,
        is_active=True
    )
    db.session.add(equipment)

equipment.equipment_status = form.equipment_status
equipment.equipment_brand = form.equipment_brand or None
equipment.equipment_model = form.equipment_model or None
equipment.boot_brand = form.boot_brand or None
equipment.boot_model = form.boot_model or None

db.session.commit()
Example SQLAlchemy — Equipment Completion Check
def is_equipment_complete(user):
    equipment = EquipmentSetup.query.filter_by(
        user_id=user.id,
        is_active=True
    ).first()

    return equipment is not None and equipment.equipment_status is not None

6️⃣ PROFILE COMPLETION LOGIC (GLOBAL RULE)
Completion is DERIVED — never stored
A profile is complete when:
is_equipment_complete(user) is true
user.primary_riding_style is not null
Example SQLAlchemy — Profile Completion
def is_profile_complete(user):
    return (
        is_equipment_complete(user)
        and user.primary_riding_style is not None
    )
7️⃣ PROGRESS GAUGE (DERIVED)
Total steps: 2
Do not store step counters
def profile_completion_progress(user):
    completed = 0

    if is_equipment_complete(user):
        completed += 1

    if user.primary_riding_style:
        completed += 1

    return completed, 2
Display as:
“Step X of 2”
Dots or small bar
