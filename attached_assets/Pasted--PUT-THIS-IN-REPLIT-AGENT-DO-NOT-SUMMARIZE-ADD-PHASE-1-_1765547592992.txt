[ My Trips ] [ Friends’ Trips ] [ Overlap ] [ Open ]

id

end_date
created_at

No resort, no state, no notes (Phase 1)

My Trips | Friends’ Trips | Overlap | Open
Rules:
Toggle switches feed content only
No route change required
Preserve existing behavior for other tabs

+ Add open dates
