## EVENT IMPLEMENTATION GUIDELINES

Prefer server-side tracking for durable product actions:

* signup_completed
* onboarding_completed
* pass_added
* availability_added
* wishlist_added
* trip_created
* friend_connected
* invite_generated

Client-side tracking is acceptable only where the action starts but may not complete, such as signup_started.

Use existing user identifiers consistently.

Do not log PII.

Do not include:

* Email
* Full name
* Phone number
* Precise location
* Free-text notes

Include useful non-PII properties where available, such as:

* source_page
* action_source
* resort_id
* trip_id
* pass_type
* rider_type
* invite_type
* is_group_trip
* friend_count
* onboarding_step
* platform
* environment

---

## RECOMMENDED EVENT FIRING POINTS

Audit and confirm the right code locations, but likely targets are:

signup_completed:

* After successful account creation and database commit

onboarding_completed:

* After the final onboarding step is saved and user is routed into the app experience

pass_added:

* When user saves ski pass / pass preference during onboarding or profile/settings

availability_added:

* When user successfully saves at least one availability date/range

wishlist_added:

* When user successfully adds a resort/mountain to wishlist

trip_created:

* After successful SkiTrip creation and database commit

friend_connected:

* After a Friend relationship is successfully created or accepted

invite_generated:

* When a friend invite link or trip invite link is successfully generated

---

## DEDUPING / DATA QUALITY

Prevent duplicate event spam where possible.

Examples:

* Do not fire pass_added repeatedly if the same pass is saved unchanged
* Do not fire availability_added if no dates were added
* Do not fire wishlist_added for duplicate wishlist entries
* Do not fire trip_created until the trip commit succeeds
* Do not fire friend_connected multiple times for the same pair
* Do not fire invite_generated repeatedly on page load; only when a link is generated or shared intentionally
