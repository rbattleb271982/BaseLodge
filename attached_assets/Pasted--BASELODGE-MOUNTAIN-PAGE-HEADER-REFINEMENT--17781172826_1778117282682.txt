║ BASELODGE MOUNTAIN PAGE HEADER REFINEMENT         ║
╚════════════════════════════════════════════════════╝

Refine ONLY the top/header section of the mountain detail page.

Goal:
Reduce vertical waste, improve information hierarchy, and make the page feel more like a destination surface without adding clutter or redesigning the rest of the page.

IMPORTANT:
- Do NOT redesign the trip list/feed section below
- Do NOT add cards, hero images, dashboards, widgets, or gamification
- Do NOT add mountain conditions/weather/snow data
- Preserve the current BaseLodge visual language:
  - cream background
  - maroon accents
  - editorial/Apple-like minimalism
  - restrained typography
  - calm spacing

────────────────────────────────────
CURRENT PROBLEM
────────────────────────────────────

Right now the top section stacks too many parallel facts vertically:

- Colorado eyebrow
- Beaver Creek
- Not on your pass
- 1 friend going

This creates:
- too much empty vertical space
- weak hierarchy
- unnecessary stacking
- page feels sparse instead of intentional

We want:
- tighter information grouping
- horizontal rhythm
- stronger destination-page feeling
- less wasted space
- more immediate content density

WITHOUT making the page busy.

────────────────────────────────────
NEW STRUCTURE
────────────────────────────────────

Remove the standalone state eyebrow entirely.

Replace:

COLORADO
Beaver Creek

with:

Beaver Creek, CO

Then compress the contextual metadata into ONE horizontal line directly beneath the title.

Example structure:

Beaver Creek, CO

Not on your pass · 1 friend going · 12 trips planned

────────────────────────────────────
DESIGN RULES
────────────────────────────────────

Title:
- remains large and editorial
- preserve current font styling/weight
- maintain strong visual presence

Metadata row:
- muted typography
- same typographic rhythm already used elsewhere in BaseLodge
  (example: “Skier · Advanced · Epic”)
- use interpuncts (·) between facts
- no pills
- no badges
- no icons
- no bold treatment
- no extra dividers

Spacing:
- reduce vertical gap above the trip list
- allow the trip list to begin higher on the page
- preserve breathing room around the title itself

────────────────────────────────────
WHAT NOT TO DO
────────────────────────────────────

Do NOT:
- add conditions/snow/weather
- add cards
- add new UI components
- add avatar stacks
- add accent bars
- add dashboard styling
- add multiple metadata rows
- redesign the trip rows below

The fix should come from:
- compression
- hierarchy
- rhythm
- typography

NOT from adding more UI.

- trip list begins noticeably higher on the page
- title maintains visual prominence
