Require an environment variable:
ADMIN_FEEDBACK_EMAIL

2. Bottom Navigation
Add a 5th bottom nav item:
- Label: Feedback
- Icon: lightbulb (outline style, consistent with existing nav icons)

Ensure the bottom nav continues to function correctly on all screens.

Some things are still a work in progress — and that’s intentional.
If you have a suggestion or notice something that could be better, we’d love to hear it.”

Form:
