Goal

Current behavior

New behavior
Move this information inside the profile card, directly under the user’s name.
Requirements
Compute next trip as the trip with the earliest start_date >= today
Display:
Countdown text (e.g. “Next trip in 13 days”)
Trip location (city + state)

Do not
Create new routes

Apply consistent tap behavior for all list-based items in the following tabs:
Tabs affected
My Trips
Friends’ Trips
Trip Overlaps
Open Availability
Rules
Entire card / row is tappable
No small or isolated tap targets
Tapping behavior:
Trips (any tab) → Edit Trip

Notes
Keep existing labels (e.g., “Edit”) for now as visual affordances
