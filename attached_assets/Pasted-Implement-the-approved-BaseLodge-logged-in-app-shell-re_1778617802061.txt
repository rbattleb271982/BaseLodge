Implement the approved BaseLodge logged-in app-shell refactor for a persistent fixed top header.

Goal:
Make BaseLodge feel more like a native app and less like a scrolling website by introducing a fixed top app bar across logged-in screens while preserving the existing fixed bottom nav. Only the content area should scroll.

Important:
This is NOT a redesign.
This is an app-shell/layout refactor only.
Use checkpoints sparingly.

Implementation requirements

Inside <div class="app-shell">, immediately after the opening tag and before {% block flash %}, add a persistent top bar.

Requirements:

- Right side:
  - Existing profile initials bubble behavior
  - Link to url_for('profile')
- Keep it compact and app-like
- Logged-in screens only
- Do not affect logged-out/public templates

Add:

Requirements:
