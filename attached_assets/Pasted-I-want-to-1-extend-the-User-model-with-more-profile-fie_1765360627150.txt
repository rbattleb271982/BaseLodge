PART 5 — Update /edit_profile route (app.py)
========================================

In app.py, update the /edit_profile route to handle POST submissions:

1. Locate the existing /edit_profile route. Modify it to accept both GET and POST:

    @app.route("/edit_profile", methods=["GET", "POST"])
    def edit_profile():
        if "user_id" not in session:
            return redirect(url_for("auth"))

        user = User.query.get(session["user_id"])
        if not user:
            session.pop("user_id", None)
            return redirect(url_for("auth"))

        if request.method == "POST":
            user.gender = request.form.get("gender") or None
            birth_year_raw = request.form.get("birth_year")
            user.birth_year = int(birth_year_raw) if birth_year_raw else None
            user.rider_type = request.form.get("rider_type") or None
            user.pass_type = request.form.get("pass_type") or None
            user.home_state = request.form.get("home_state") or None
            user.skill_level = request.form.get("skill_level") or None

            db.session.commit()
            return redirect(url_for("profile"))

        # for GET, render edit_profile.html with current user
        return render_template("edit_profile.html", user=user)

Make sure `request` is imported at the top of app.py:

    from flask import request

And ensure db and User are already imported.
