ROUTE REQUIREMENTS
--------------------------------------------------

In the existing admin_dashboard view, add a small block of read-only queries after the existing dashboard calculations and before render_template.

Compute and pass:

activation_pct
= is_active_user / non-seeded total users

invite_redemption_pct
= InviteToken used_at is not null / InviteToken generated by non-seeded users

top_state
= most common User.home_state among non-seeded users with a state

top_state_pct
= top_state count / users with a home_state

avg_product_score
= same formula as Product Dashboard

Important:

- Do not change existing vars.
- Do not change existing KPI logic.
- Do not alter current dashboard query behavior.
- Add only new vars needed for the new hub sections.
