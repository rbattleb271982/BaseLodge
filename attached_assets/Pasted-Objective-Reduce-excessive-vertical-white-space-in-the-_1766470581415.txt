Objective
Reduce excessive vertical white space in the profile card while preserving readability, hierarchy, and tap targets.
This is a spacing + typography tuning pass only.
Do NOT change layout structure, copy, or functionality.

⸻

Scope
Apply spacing adjustments to:
	•	User profile card
	•	Friend profile cards
	•	Any surface where the profile card appears

All changes must apply via the shared profile card partial/component.

⸻

Required spacing adjustments (exact)
	1.	Name → Hero line
Reduce vertical spacing between the user name and the hero line (Skier · Pass · Skill).

	•	Target spacing: approximately 8–10px
	•	Remove any large bottom margin currently applied to the name

⸻

	2.	Hero line → Trip summary
Reduce spacing between the hero line and the “X upcoming trips” line.

	•	Target spacing: approximately 6–8px
	•	Should feel like a tight informational grouping, not headline → paragraph

⸻

	3.	Equipment + Boots grouping
Treat Equipment and Boots as a single grouped block, not separate sections.

	•	Reduce spacing above Equipment to ~8px
	•	Reduce spacing between Equipment and Boots to ~4px
	•	Do NOT reduce font size
	•	Do NOT change copy or color

⸻

	4.	Divider spacing
Tighten the vertical padding around the divider separating profile details from mountain rows.

	•	Padding above divider: ~8px
	•	Padding below divider: ~8px
	•	Divider remains visually present; do not remove it

⸻

	5.	Mountain rows internal padding
Reduce vertical padding inside each mountain row.

	•	Target padding top/bottom: ~10–12px
	•	Ensure rows remain clearly tappable
	•	Remove any extra margin between the two rows so they read as a compact list

⸻

Constraints (important)
	•	Do NOT reduce tap target height below accessibility minimums
	•	Do NOT reduce font sizes
	•	Do NOT change colors
	•	Do NOT remove sections
	•	Do NOT alter modal behavior
	•	Do NOT introduce new layout containers

This is strictly a spacing refinement pass.

⸻

Acceptance criteria
	•	Profile card height is reduced by approximately 20–30%
	•	Visual hierarchy remains clear
	•	Card still feels calm, readable, and premium
	•	No layout regression on mobile
	•	No tap targets feel cramped
	•	No console or layout errors

⸻

Testing required (do not skip)
	•	Test self profile and friend profile
	•	Test users with and without equipment
	•	Test users with 0 mountains and 40+ mountains
	•	Verify card height reduction is visually noticeable
	•	Verify no text overlap or clipping

Do not finish until all tests pass.
