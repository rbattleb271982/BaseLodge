#172 - Admin Dashboard Navigation Cleanup

What & Why

The admin suite now has several major intelligence dashboards live:

- Growth
- Activation
- Retention
- Resort Intelligence
- Product
- Messaging
- User Intelligence

The navigation is getting crowded and needs to be organized so the admin panel feels intentional, not like a long list of pages.

Goal:

Create a cleaner admin navigation structure that groups related dashboards together and makes the admin console easier to scan on desktop and mobile.

Do not change dashboard logic.

Do not change routes.

Do not change metrics.

This is navigation/UI organization only.

--------------------------------------------------
RECOMMENDED NAV GROUPS
--------------------------------------------------

Group admin links into clear sections:

1. Overview

- Dashboard
- Growth
- Activation
- Retention

2. Intelligence

- User Intelligence
- Product
- Resort Intelligence
- Messaging

3. Operations

- Trips
- Users
- Tools
- Feedback
- Settings / Diagnostics if present

Use existing routes.

Do not rename URLs.

--------------------------------------------------
DESKTOP REQUIREMENTS
--------------------------------------------------

On desktop:

- Use grouped navigation headings
- Keep links readable
- Highlight active section/page
- Preserve existing anchor sub-links where applicable
- Avoid making the nav feel visually heavy

--------------------------------------------------
MOBILE REQUIREMENTS
--------------------------------------------------

On mobile:

- Navigation must remain usable
- Groups should collapse cleanly if needed
- No horizontal overflow
- No clipped nav text
- Current page should still be obvious

--------------------------------------------------
DESIGN REQUIREMENTS
--------------------------------------------------

Match existing admin visual style.

Use existing colors, spacing, and typography.

Do not introduce a new design system.

--------------------------------------------------
QA REQUIREMENTS
--------------------------------------------------

Verify:

- All existing admin links still work
- Active states work
- Anchor sub-links still work
- Desktop nav looks clean
- Mobile nav is usable
- No template errors
- No route changes
- No metric/query changes

Use checkpoints sparingly.
