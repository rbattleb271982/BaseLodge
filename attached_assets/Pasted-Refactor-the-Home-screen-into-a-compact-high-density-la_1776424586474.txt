Refactor the Home screen into a compact, high-density layout using horizontal structure instead of tall stacked cards.

This is NOT a styling tweak.
This is a layout system change.

GOALS
1. Reduce vertical height across all cards
2. Eliminate heavy card backgrounds (no pink, no white boxes)
3. Use horizontal layout (columns) instead of vertical stacking
4. Right-align all CTAs consistently
5. Make the screen scalable for 20–30+ items

--------------------------------------------------

1. REMOVE CARD BACKGROUNDS

- Remove background colors from:
  - Trip card
  - Invite banner
  - Best Match card

- Replace with:
  - Transparent background
  - Use subtle separators instead:
    - border-top OR border-bottom: 1px solid #E5DFD0

- The page should feel like a clean list, NOT stacked boxes

--------------------------------------------------

2. CONVERT ALL CARDS TO ROW-BASED LAYOUT

Each item should behave like a structured row, not a block.

--------------------------------------------------

- No extra vertical spacing

- Reduce spacing between label and content
- No extra padding

--------------------------------------------------

9. SPACING SYSTEM

- Reduce vertical spacing between rows
- Use consistent gap (8–12px max)
- Remove large empty padding blocks

--------------------------------------------------

10. MOBILE VALIDATION

Ensure:
- No wrapping breaks layout awkwardly
- Long resort names truncate cleanly
- Buttons stay aligned right
- Rows remain compact

--------------------------------------------------

11. DO NOT

- Do not reintroduce colored backgrounds
- Do not create large card containers again
- Do not stack content vertically if horizontal space is available

--------------------------------------------------

OUTPUT

- Confirm all elements converted to row-based layout
- Confirm CTAs are right-aligned across the page
