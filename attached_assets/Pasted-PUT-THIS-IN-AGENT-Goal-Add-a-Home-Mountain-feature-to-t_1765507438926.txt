Goal: Add a “Home Mountain” feature to the BaseLodge app.

This requires:
1. Adding a new column to the User model
2. Creating a migration with Flask-Migrate
3. Updating Edit Profile UI to include a Home Mountain dropdown
4. Updating Add Trip UI to offer a “Use My Home Mountain” quick button and prioritize the home mountain in the dropdown list

----------------------------------------
STEP 1 — UPDATE USER MODEL
----------------------------------------
File: models.py (where User model is defined)

Add this field inside the User model class:

    home_mountain = db.Column(db.String(100), nullable=True)

Do not modify other fields.

----------------------------------------
STEP 2 — CREATE MIGRATION
----------------------------------------
Run:

flask db migrate -m "Add home_mountain field to User"
flask db upgrade

STEP 5 — UPDATE ADD TRIP PAGE
----------------------------------------
File: templates/add_trip.html

(1) Below the <h1>Add a trip</h1> insert:

{% if current_user.home_mountain %}
<button id="useHomeMountain" class="btn-primary" style="margin-bottom: 1rem;">
    🏠 Use My Home Mountain ({{ current_user.home_mountain }})
</button>
{% endif %}

(2) Add JS at the bottom of the file before </script>:

const userHomeMountain = "{{ current_user.home_mountain or '' }}";

document.getElementById("useHomeMountain")?.addEventListener("click", () => {
    if (!userHomeMountain) return;

    // Find correct state based on mountain dictionary
    for (const [state, mountains] of Object.entries(MOUNTAINS)) {
        if (mountains.includes(userHomeMountain)) {
            document.getElementById("state").value = state;
            loadMountains();
            document.getElementById("mountain").value = userHomeMountain;
            break;
        }
    }
});

(3) Modify loadMountains() so that when mountains populate, if the user's home mountain is in the list, place it first:

Within mountains.forEach(), replace with:

mountains.forEach(m => {
    const option = document.createElement("option");
    option.value = m;
    option.text = (m === userHomeMountain) ? m + " (Home Mountain)" : m;
    mountainSelect.appendChild(option);
});
