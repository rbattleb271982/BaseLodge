Goal
Implement a complete, low-friction onboarding + post-login guidance flow for BaseLodge that:
Captures core identity on signup (single screen, no scroll)
Progressively completes the profile after login (modal pop-ups with a step gauge)
Ends with a one-time “Welcome to BaseLodge” orientation screen that tells users what to do next
Never nags users after the second login
This must work for renters and non-experts and must not block app usage.

2️⃣ POST-LOGIN PROGRESSIVE PROFILE COMPLETION
When to show
On 1st login
On 2nd login only if profile is incomplete
Never show after that
Completion definition (derived, not stored)
Profile is “complete” when:
equipment_status IS NOT NULL
AND
primary_riding_style IS NOT NULL
3️⃣ POP-UP FLOW STRUCTURE
Rules (important)
Each answer saves immediately to the User model
No “submit at the end”
Skipping never clears data
Progress gauge is derived, not stored
Step gauge
Total steps: 2
Display as “Step X of 2” with dots or small bar
Gauge reflects completed fields, not modal order
